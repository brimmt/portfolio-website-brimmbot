What Is It?

Proposal AI is an intelligent assistant designed to help businesses create polished, on-brand proposals with minimal effort. It uses natural language generation to draft customized proposals based on client needs, project goals, and tone of voice.

Built for speed and professionalism, the system integrates with form inputs and knowledge sources to auto-generate proposal documents that can be reviewed, edited, and sent by a human with minimal revisions.

Why I Built It

So many businesses spend hours writing proposals from scratch — or worse, copying from old templates that don’t reflect their best work.

I built Proposal AI to save time and bring quality to the forefront. It allows teams to respond faster, stay on-brand, and win more opportunities without being bogged down by repetitive writing.

Tech Stack

  • 🧠 OpenAI API — for natural language generation
  • ⚙️ FastAPI + Python — backend and API logic
  • 🧩 HTML + Tailwind CSS — frontend interface
  • 🗃️ Supabase — storing client templates and outputs
  • 📨 Email & PDF Integration — for sharing polished documents
