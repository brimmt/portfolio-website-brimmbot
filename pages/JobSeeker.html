What Is It?

Job Seeker AI is a smart assistant that helps users organize their job search in a focused, AI-enhanced workspace.

It includes a customized interface for generating resumes, cover letters, job-specific tailoring, and even interview prep — all powered by natural language and prompt-based workflows.

Why I Built It

Job searching can feel overwhelming — and most people don’t have a system in place to organize applications, tailor resumes, or prep for interviews effectively.

I built Job Seeker AI to bring structure, automation, and support to that process. I wanted something that felt personal, efficient, and actionable — not just another resume template site.

Tech Stack

  • 🧠 OpenAI API — for cover letters, resumes, and interview prep generation
  • ⚙️ FastAPI + Python — for backend and form logic
  • 🗃️ Supabase — for job tracker and user submission storage
  • 🎨 Tailwind CSS — for a modern, responsive UI
  • 📄 Jinja2 / HTML — for rendering dynamic job materials
