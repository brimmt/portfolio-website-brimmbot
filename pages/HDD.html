Healthcare Data Dashboard (Coming Soon)

What Is It?

This project started as a deep data cleaning exercise using a large healthcare dataset — but quickly evolved into something more powerful.

Instead of just fixing messy columns and inconsistencies, I transformed the dataset into a usable system that supports real insights and decision-making.

The current version includes a dynamic frontend dashboard to help users easily filter, sort, and analyze patient records. It’s also being enhanced with AI-powered tools for pattern detection, summarization, and workflow suggestions.

Why I Built It

In healthcare, messy data isn’t just inconvenient — it leads to poor decisions and lost time. I wanted to take a realistic dataset and show how we could go from chaos to clarity.

Rather than stopping at cleaning, I designed a system that lets users actually work with the data — including a visual interface, smart filters, and soon, AI integrations for automatic insights.

This project reflects my mission to bridge backend logic with frontend usability — especially for industries like healthcare, where data can truly make a difference.

Tech Stack

  • 📊 Microsoft Excel — for data cleaning, transformation, and early analysis
  • 📈 Tableau — for interactive dashboard design and KPI tracking
  • ⚙️ Microsoft Power BI — for advanced visualizations and report building
  • 🧪 Jupyter Notebook (V2) — for statistical analysis and AI forecasting models
  • 🚀 FastAPI (V2) — for building a lightweight backend and serving AI endpoints

Why I Chose This Stack

Excel, Tableau, and Microsoft PowerBI are still some of the most commonly used tools in hospital analytics. I wanted to show that I could not only use them — but use them strategically.

I chose this stack for accessibility, speed, and real-world application — with plans to add forecasting models in Python for Version 2.
