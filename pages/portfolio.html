Personal Portfolio Website

🚀 Live Site

What Is It?

This is my personal portfolio site — a curated space to showcase my projects, skills, and personality. It’s a digital home for recruiters, collaborators, and friends to explore who I am and what I build.

Why I Built It

I wanted a full-stack project that felt meaningful. Not just a GitHub repo — but a real experience that reflects my journey.
This site became my digital home — a place where I could host all my projects, link to live demos, and build tools like BrimmBot to guide people through what I’m creating.

I wanted recruiters, clients, and friends to explore not just my work — but the way I think, build, and grow as a developer.

Tech Stack

  • 🧩 HTML + Tailwind CSS — for styling and layout
  • ⚙️ JavaScript (vanilla) — for interactive elements
  • 🐍 Python + FastAPI — for backend APIs
  • 🧠 OpenAI API — for future AI content and assistants
  • 🕹️ n8n — for low-code automation

Why I Chose This Stack

I chose this stack because I want to become more proficient in these tools. I’m actively working toward becoming a full-stack developer, and building a portfolio site like this gave me a real opportunity to test what I know — while also learning new things along the way.
